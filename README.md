## 📓 겪언던 문제, 느낌점, 새로 알게된 정보

### 라이브러리의 간편함
솔직히 내 실력에 Drag & Drop을 쌩으로 구현하려면 상당히 많은 시간이 소요됐을 것이다.<br/>
하지만 **dnd-kit** 라이브러리 덕분에 빠른 시간 안에 기능을 구현할 수 있었다.

### OAuth의 무책임함
OAuth 덕분에 나는 이용자들의 개인정보같은 민감한 정보들을 저장할 필요가 없다.<br/>
그 말인 즉슨 내게 개인정보에 대한 책임은 없는 것이다 으하하.<br/>
그리고 내가 구현해야 하는 로그인 기능도 줄어든다.<br/>
라이브러리라는 것은 좋은 것같다.
